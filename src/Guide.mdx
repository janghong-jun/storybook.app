# 하이코칭 디자인 시스템 프로젝트 가이드

<div style={{ marginBottom: 16 }} />

이 문서는 프로젝트에서 디자인 시스템 컴포넌트를 쉽게 확인하고 바로 적용할 수 있도록  
필요한 정보만 정리한 가이드입니다. Storybook 기준으로 컴포넌트 구조, 사용 방법, 속성 등을 확인할 수 있습니다.

<div style={{ marginTop: 32, marginBottom: 32 }}>
  <hr />
</div>

## 📚 문서에서 확인할 수 있는 내용

<div style={{ marginBottom: 12 }} />

- 버튼, 입력 요소, 아코디언 등 기본 컴포넌트의 Story와 구조  
- 각 컴포넌트의 Props, Controls, 상태 변화  
- 실제 화면 제작 시 참고할 수 있는 마크업·사용 예시

<div style={{ marginTop: 32, marginBottom: 32 }}>
  <hr />
</div>

## 🚩 Storybook 사용 방법

<div style={{ marginBottom: 8 }} />

1. 왼쪽 메뉴에서 필요한 컴포넌트를 선택합니다.  
2. Props를 조절하면서 실제 UI가 어떻게 변하는지 확인합니다.  
3. Story에 포함된 마크업과 예시 코드를 그대로 가져다 쓰면 가장 빠릅니다.

<div style={{ marginTop: 32, marginBottom: 32 }}>
  <hr />
</div>

## 💡 실제 작업할 때 참고할 점

<div style={{ marginBottom: 8 }} />

- Storybook에 있는 구조가 기본 기준입니다.  
- 새로운 페이지 작업할 때는 기존 컴포넌트를 우선 활용하는 게 좋습니다.  
- 디자인 시안과 미묘하게 다를 경우 Storybook 기준을 맞춰주세요.  
- 확장이 필요한 경우 기존 패턴을 유지하면서 props 또는 스타일을 추가하는 방식으로 진행합니다.
